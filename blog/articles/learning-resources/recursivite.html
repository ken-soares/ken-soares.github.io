1. Récursivité

définition: Une fonction est dite récursive si elle fait appel à elle-même.

1.1. Règles importantes concernant l'utilisation de la récursivité

  1. Une fonction récursive doit contenir une ou des conditions d'arrêt (pour éviter que le programme tourne indéfiniment).
  2. Les valeurs passées dans les appels récursifs doivent être différentes de celles utilisées dans les appels précédents. (par exemple, une fonction foo(bar) peut appeller foo(bar-1)).
  3. Après un certain nombre fini d'appels, la ou les valeurs passées en paramètre doivent permettre de satisfaire une des conditions d'arrêt.
  4. Le nombre d'appels récursifs ne doit pas dépasser la profondeur de récursivité (en Python, cette dernière est aux alentours de 1000).

1.2. Récursivité simple ou multiple

Lorsque chaque appel de fonction engendre au plus un appel récursif, on parle de récursivité simple mais on peut rencontrer une récursivité multiple où chaque appel de fonction peut effectuer plusieurs appels récursifs.

Attention: Avec les récursivités multiples, le nombre d'appels récursifs peut être exponentiel et vite "exploser" (ce qui violerait la règle 4).

2. Application concrète avec la fonction factorielle

def facto(n):
  if n == 1:
    return 1
  else:
    return n * facto(n-1)
